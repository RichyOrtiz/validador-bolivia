# Validador de Billetes Serie B - Bolivia 🇧🇴

Aplicación PWA para la verificación de billetes inhabilitados de la Serie B.

### Características:
* 📱 **Instalable:** Funciona como App en Android e iOS.
* 📷 **Escáner:** Validación mediante cámara.
* 🚨 **Alertas:** Sonido de sirena y parpadeo rojo para billetes inhabilitados.
* 📶 **Modo Offline:** Funciona sin conexión a internet.

### Uso:
1. Abrir la App.
2. Seleccionar el corte (10, 20 o 50 Bs).
3. Ingresar la serie o escanear.
4. Si la pantalla parpadea en rojo y suena la alarma: **¡NO RECIBIR!**
